# ASCII Ray Tracing

A simple ray tracer inspired by the "Ray Tracing in one weeked" book, which display the rendered scene using ascii.

## Installation

```shell
git clone https://github.com/SlowlyCoding/ascii-ray-tracer
```
```shell
cd ascii-ray-tracer
```
```shell
make
```
